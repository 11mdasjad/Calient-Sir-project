## 🚀 Getting Started

### 1. **Open Locally** (1 minute)
```bash
# Double-click index.html
# Or use local server:
python3 -m http.server 8000
# Visit: http://localhost:8000
```

### 2. **Customize** (15 minutes)
- Update company name
- Change colors (CSS variables)
- Add your content
- Update contact info
- Add images

### 3. **Deploy** (5 minutes)
**Option A - Netlify (Easiest)**
```
1. Visit netlify.com
2. Drag & drop folder
3. Done! You have a live site
```

**Option B - GitHub Pages**
```
1. Create GitHub repo
2. Upload files
3. Enable Pages
4. Live at yourusername.github.io/fintech-website
```
